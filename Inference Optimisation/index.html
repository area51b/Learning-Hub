Inference Optimisation

A structured curriculum for understanding modern transformer-based LLM serving systems - from model internals to fleet-scale infrastructure

28 modules across 5 parts. Each module is a standalone ~10 min read with expandable sections, interactive elements, and a quiz. Start at Part 1 or jump to any module directly. Parts 1–2 cover the model-level concepts that everything else builds on. Parts 3–4 cover per-instance optimisation. Part 5 covers fleet-scale coordination with NVIDIA Dynamo and llm-d.
